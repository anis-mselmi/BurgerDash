# BurgerDash

<img src="assets/background.png" alt="Background" width="500"/>

Bienvenue dans **BurgerDash**, un jeu rapide et amusant de restauration rapide !

## Description

Dans BurgerDash, vous incarnez un chef de restaurant. Les clients arrivent avec des commandes spécifiques (burgers, frites ou boissons) et vous devez leur servir le bon plat avant que le temps ne s'écoule. Plus vous êtes rapide, plus votre score augmente. Soyez vigilant, la moindre erreur ou le temps écoulé signifie la fin de la partie !

## Comment jouer

1. Assurez-vous d'avoir Python et la bibliothèque Pygame installés sur votre machine.
2. Lancez le jeu en exécutant la commande suivante dans votre terminal :
   ```bash
   python main.py
   ```
3. Cliquez sur l'image du plat en bas de l'écran qui correspond à la commande du client.

Amusez-vous bien !
